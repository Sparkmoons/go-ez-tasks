## Вывод числа Фибоначчи

Определяется n-ое число последовательности Фибоначчи.

## Реализация

```go
func FibNumb(n int) int{
  if n < 2 {
    return n
  }
  var a, b int
  a, b = 0, 1
  for n--; n > 0; n-- {
    a, b = b, a + b
  }
  return b
}
```

## Пример
Пусть n = 1. На выводе будет: 1
Пусть n = 8. На выводе будет: 21
